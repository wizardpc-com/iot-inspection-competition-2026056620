# 仪表检测模型接口说明

## 模型文件

- `meter_best.pt`：YOLOv5s 训练 100 轮，样例 mAP 0.869。
- `meter_last.pt`：可选训练检查点。

## 原始使用方式

```bash
python detect.py --weights best.pt --source 图片路径 --save-txt --save-conf
```

## 输出

- 检测结果图：绘制检测框的图片。
- 检测结果 txt：包含类别 ID、中心点坐标、宽高和置信度。

## 当前能力

当前模型可检测仪表盘区域，并框出指针、刻度等关键部件。ROS2 接入后，`meter_detector_node` 会发布检测框、类别、置信度、统计结果、标注图路径和估算读数。

## 读数方法

系统基于 `base/start/end/tip` 的检测框中心点估算指针位置，并按照配置量程输出 `reading_value`：

- `base`：表盘角度中心
- `start`：起始刻度位置
- `end`：结束刻度位置
- `tip`：指针尖端位置

读数精度依赖关键部件检测质量、量程配置和角度方向配置。
