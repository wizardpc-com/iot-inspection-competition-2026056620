# 仪表模型接口说明

## 模型文件

```text
models/meter_best.pt
```

该权重为 YOLOv5 旧格式模型，运行时需要本地 YOLOv5 工程：

```text
models/yolov5/detect.py
```

## ROS2 节点

```text
meter_detector_node
```

节点订阅：

```text
/inspection/image_path
```

节点发布：

```text
/vision/meter_result
```

## 输入

`/inspection/image_path` 使用 `std_msgs/String`，内容为 JSON：

```json
{
  "station_id": "P001",
  "image_path": "/abs/path/demo_images/meter_01.jpg",
  "timestamp": "2026-05-06T10:00:00",
  "source_type": "demo_image"
}
```

## 输出

`/vision/meter_result` 使用 `std_msgs/String`，内容为 JSON。`model_backend` 为 `yolov5_detect_py`。主要字段包括：

- `station_id`
- `image_path`
- `annotated_image_path`
- `annotated_save_ok`
- `detected`
- `detector_type`
- `model_backend`
- `count`
- `class_counts`
- `max_conf`
- `detections`
- `meter_status`
- `reading_status`
- `reading_value`
- `reading_unit`
- `reading_ratio`
- `reading_method`
- `reading_reason`
- `reason`
- `timestamp`
- `error`

## 读数估算方法

当前实现使用 `base/start/end/tip` 检测框中心点估算读数：

1. 以 `base` 的中心点作为角度中心。
2. 计算 `start`、`end`、`tip` 相对 `base` 的角度。
3. 根据 `meter_angle_direction` 计算 `tip` 在 `start` 到 `end` 之间的比例。
4. 将比例映射到 `meter_min_value` 和 `meter_max_value`。

配置文件：

```text
ros2_ws/src/inspection_mvp/config/demo.yaml
```

该方法适合当前比赛原型演示。若后续接入真实仪表，需要根据仪表量程和刻度关系进行标定。
